# Dark++ Regular Theme
Enhanced VScode Dark+ theme.

![Theme Overview](overview.png)

### Fira Code
1. Add following options in User Settings:
    ```
    {
      ...
      "editor.fontFamily": "Fira Code",
      "editor.fontLigatures": true,
      ...
    }
    ```
2. Reload VSCode window or reopen VSCode.
